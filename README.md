# ARM-CortexM4-HelloWorld

## Requirements 

### Hardware
* STM32F407G-DISC1 (Development Kit)
* Micro-USB to USB-A
### Software
* STM32Cube IDE

### Hardware Specifiations
* Core Architecture - Arm
* Data Bus Width - 32bits
* Memory Size - 1MB
* RAM Size - 192kB
* Interface - CAN, Ethernet, I2C, SPI, UART, USART and USB
